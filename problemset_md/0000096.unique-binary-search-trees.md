Given _n_, how many structurally unique **BST's** (binary search trees) that store values 1 ... *n*?

**Example:**

<pre><strong>Input:</strong> 3
<strong>Output:</strong> 5
<strong>Explanation:
</strong>Given <em>n</em> = 3, there are a total of 5 unique BST's:

   1         3     3      2      1
    \       /     /      / \      \
     3     2     1      1   3      2
    /     /       \                 \
   2     1         2                 3
</pre>

**Constraints:**

- `1 <= n <= 19`
