Write an algorithm such that if an element in an MxN matrix is 0, its entire row and column are set to 0.

**Example 1:**

<pre><strong>Input: </strong>
[
  [1,1,1],
  [1,0,1],
  [1,1,1]
]
<strong>Output: </strong>
[
  [1,0,1],
  [0,0,0],
  [1,0,1]
]
</pre>

**Example 2:**

<pre><strong>Input: </strong>
[
  [0,1,2,0],
  [3,4,5,2],
  [1,3,1,5]
]
<strong>Output: </strong>
[
  [0,0,0,0],
  [0,4,5,0],
  [0,3,1,0]
]
</pre>
