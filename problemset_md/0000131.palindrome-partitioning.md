Given a string _s_, partition _s_ such that every substring of the partition is a palindrome.

Return all possible palindrome partitioning of _s_.

**Example:**

<pre><strong>Input:</strong>&nbsp;"aab"
<strong>Output:</strong>
[
  ["aa","b"],
  ["a","a","b"]
]
</pre>
